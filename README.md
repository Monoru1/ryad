# RS — Premium Engineering Portfolio

Portfolio cinématographique construit avec Next.js, TypeScript, TailwindCSS, Framer Motion, Three.js et Lenis.

## Stack

- Next.js
- TypeScript
- TailwindCSS
- Framer Motion
- Three.js
- React Three Fiber
- GSAP
- Lenis Scroll

---

# Installation

```bash
npm install
```

---

# Développement local

```bash
npm run dev
```

---

# Build production

```bash
npm run build
```

---

# Déploiement Vercel

## Méthode rapide

1. Push du repo sur GitHub
2. Aller sur https://vercel.com
3. Import Git Repository
4. Sélectionner `portefolio-rs`
5. Deploy

Configuration automatique détectée.
